## 目前支持的命令

1. **init**

   - 初始化项目，从 GitHub 上下载

2. **-V**

   - 查看版本

3. **--help**

   - 查看可用指令

4. **ls**
   - 查看所有项目模板

## 项目简介

1. Vue3+TS 的项目脚手架

   - **基于 webpack 构建生成**
   - **集成:eslint、prettier、stylelint、husky、tailwindcss、scss、pinia、vue-router**

2. Webpack+TS 的项目脚手架
   - **一些基于 webpack+ts 的项目配置**

3. monorepo预设
   - **基于pnpm+pnpm-workspace的monorepo预设**
